# 信息安全与保密制度（2026修订版）

## 1. 信息安全总则

### 1.1 适用范围
- 本制度适用于全体员工、实习生、外包人员
- 所有人员入职时须签署《保密承诺书》
- 离职时须完成信息安全交接

### 1.2 保密等级划分
| 等级 | 标识 | 内容范围 | 示例 |
|------|------|---------|------|
| 绝密 | 🔴绝密 | 核心商业机密 | 战略规划、融资信息 |
| 机密 | 🟠机密 | 重要商业信息 | 客户数据、财务数据 |
| 秘密 | 🟡秘密 | 一般内部信息 | 薪酬体系、内部制度 |
| 内部 | 🔵内部 | 仅限员工知晓 | 组织架构、待发布通知 |
| 公开 | ⚪公开 | 对外可公开 | 官网内容、公开新闻 |

### 1.3 违规处罚
- 泄露绝密/机密信息：立即解除劳动合同，追究法律责任
- 泄露秘密/内部信息：记大过一次，情节严重者解除合同
- 未造成后果的违规操作：书面警告

## 2. 密码与账号管理

### 2.1 密码安全要求
- 密码长度：至少 12 位
- 密码复杂度：必须包含大写字母、小写字母、数字、特殊字符
- 密码有效期：每 90 天强制更换
- 密码历史：不可重复使用最近 5 次的密码
- 禁止行为：密码写在便签贴在电脑上、告知他人

### 2.2 账号管理
- 员工账号与工号绑定，离职后自动封禁
- 严禁共享账号、借用他人账号
- 临时账号需在 30 天内重新申请续期
- 账号异常登录（异地 IP）系统自动告警

### 2.3 MFA 多因素认证
- 必须绑定企业微信扫码 + 手机验证码
- 以下场景强制启用 MFA：
  - 登录企业邮箱（Web 版）
  - 登录 VPN
  - 访问财务系统

## 3. 电脑与设备安全

### 3.1 设备安全要求
- 离开工位必须锁屏（Win+L / Ctrl+Cmd+Q）
- 电脑必须安装公司指定的杀毒软件（360 企业版）
- 禁止关闭防火墙
- 禁止越狱/Root 破解设备

### 3.2 软件安装
- 安装新软件需提交 IT 申请，由 IT 代为安装
- 禁止安装来路不明的软件（盗版软件、游戏外挂等）
- 禁止使用 P2P 下载软件（迅雷、电驴等）
- 浏览器仅允许使用 Chrome、Edge、Firefox

### 3.3 设备维修与报废
- 设备故障：联系 IT 部门，禁止私自拆修
- 设备报废：IT 部门进行数据清除后走报废流程
- 离职归还：所有公司设备须在离职前归还 IT 部门

## 4. 网络与数据传输

### 4.1 网络访问规范
- 访问公司内网必须通过 VPN（在家/出差场景）
- VPN 账号仅限本人使用，禁止共享
- 禁止访问赌博、色情、反动等违法网站
- 禁止使用翻墙软件

### 4.2 数据传输规范
- 公司数据只能通过公司邮箱、企业微信、公司网盘传输
- 禁止使用个人邮箱、个人网盘传输公司文件
- 向外部发送文件需进行敏感信息脱敏处理
- 大文件（超过 50MB）使用公司企业网盘分享链接

### 4.3 邮件安全
- 邮件附件包含敏感信息需加密压缩
- 邮件主题禁止直接写「绝密」「机密」等字眼
- 来历不明的邮件链接不要点击，举报给 IT
- 钓鱼邮件识别：发件人地址异常、链接域名可疑

## 5. 敏感数据保护

### 5.1 客户信息保护
- 客户联系方式、交易记录严禁外泄
- 客户名单按机密等级保护
- 未经客户书面授权，禁止将客户信息用于第三方

### 5.2 员工信息保护
- 员工薪酬属于绝密，禁止相互打听
- 人事档案仅 HR 及直属上级有权查阅
- 离职员工的个人信息按照法规要求保留或销毁

### 5.3 财务数据保护
- 财务系统访问权限最小化原则
- 财务报表未经授权不得外传
- 银行U盾、印鉴章妥善保管

## 6. 物理安全

### 6.1 访客管理
- 访客须在前台登记，领取临时访客证
- 访客须由公司员工全程陪同
- 访客不得单独进入办公区、机房等敏感区域
- 访客拍照须经被访部门负责人同意

### 6.2 门禁管理
- 门禁卡仅限本人使用，禁止转借
- 门禁卡遗失须立即报告行政
- 尾随他人进入属于违规行为
- 机房实行双人开门制度

### 6.3 文档打印与销毁
- 重要文件打印后及时取走
- 敏感文件使用碎纸机销毁
- 含有敏感信息的白板拍照记录后擦除

## 7. 应急响应

### 7.1 安全事件报告
发现以下情况须立即报告 IT 和 HR：
- 电脑中毒、勒索软件
- 账号被盗用
- 发现陌生人异常访问
- 敏感信息泄露

### 7.2 应急处置
- IT 部门 24 小时值班：拨打 8001
- 安全事件响应时效：
  - 高危：2 小时内响应
  - 中危：4 小时内响应
  - 低危：24 小时内响应
